# 🛡️ Mentari Mod v2.0 (Official Distribution)

Ekstensi Chrome untuk meningkatkan pengalaman belajar di **MENTARI UNPAM** dengan fitur pelacakan forum, asisten AI, bantuan quiz, dan manajemen presensi.

---

### 🚀 Fitur Utama

-   **🤖 Gemini AI Assistant**: Bantuan AI langsung di halaman Mentari.
-   **📚 Guidebook Mode**: Panduan belajar instan.
-   **📝 Quiz Helper**: Mempermudah pengerjaan tugas dan kuis.
-   **🕒 Presence Management**: Memastikan presensi Anda tercatat dengan benar.
-   **💬 Forum Tracking**: Memantau diskusi di forum secara otomatis.

---

### 🛠️ Cara Install (Manual)

1.  Download (Clone) repositori ini.
2.  Buka browser Chrome dan akses `chrome://extensions/`.
3.  Aktifkan **Developer Mode** di pojok kanan atas.
4.  Klik tombol **Load unpacked**.
5.  Pilih folder **`dist`** dari repositori yang sudah Anda download.
6.  Selesai! Ekstensi siap digunakan.
